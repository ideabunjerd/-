# ซุ่ม — กินอะไรดี 🥗

เว็บแอปสุขภาพภาษาไทย ช่วย **สุ่มเมนูอาหารแต่ละมื้อ** พร้อมติดตามแคลอรี่และดูแลตัวเองในแต่ละวัน — ทำงานได้ทั้งหมดในเบราว์เซอร์ ไม่ต้องมีเซิร์ฟเวอร์ และเก็บข้อมูลไว้ในเครื่องด้วย `localStorage`

## ✨ ฟีเจอร์

- **สุ่มเมนูรายมื้อ** — เช้า / กลางวัน / เย็น / ค่ำ / อาหารว่าง พร้อมตัวกรอง (คลีน / โปรตีนสูง / มังสวิรัติ) และสุ่มให้ไม่เกินเป้าแคลอรี่ของมื้อนั้น
- **ตั้งเป้าแคลอรี่แต่ละมื้อ** และดูเป้ารวมทั้งวัน
- **ติดตามสารอาหาร** โปรตีน / คาร์บ / ไขมัน (ค่าประมาณจากแคลอรี่)
- **บันทึกน้ำดื่ม** และ **แคลอรี่จากการออกกำลังกาย**
- **บันทึกอาหารรายวัน** พร้อมเวลาที่บันทึก และเปิดดูย้อนหลังได้
- **สรุปรายวัน** — แคลอรี่ที่กิน / คงเหลือ / คะแนนสุขภาพ
- **หน้าสถิติ** — กราฟ 7 วันล่าสุด + ค่าเฉลี่ย + ประวัติรายวัน
- **ธีมสว่าง / มืด / ตามระบบ**

## 🚀 วิธีใช้

เปิดไฟล์ `index.html` ในเบราว์เซอร์ได้เลย หรือเปิดผ่าน GitHub Pages

> ตัวเลขแคลอรี่และสารอาหารเป็นค่าประมาณเพื่อการอ้างอิงเบื้องต้น ไม่ใช่คำแนะนำทางการแพทย์
